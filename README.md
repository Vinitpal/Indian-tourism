## Run Locally

Clone the project

```bash
  git clone https://github.com/Vinitpal/Indian-tourism
```

Go to the project directory

```bash
  cd indian-tourism
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
